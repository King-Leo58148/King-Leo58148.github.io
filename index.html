About

I'm a Computer Science student at KNUST who spends more time building than reading about it. My GitHub is where I think out loud — through REST APIs, real-time systems, and cloud-integrated backends built from scratch.

My projects include a secure photo management API with Cloudinary storage and Redis caching, a real-time group chat system using Django Channels and WebSockets, and a collaborative e-commerce backend — all designed with production principles in mind.

My stack is Python and Django at the core, with MySQL, PostgreSQL, Redis, and Django REST Framework as daily tools. I'm expanding into Spring Boot as I push toward full backend versatility.

Always building.
