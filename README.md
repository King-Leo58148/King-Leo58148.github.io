# Louis Aparkwon Sanyenni — Portfolio

This is a simple dark-themed personal portfolio built with HTML, CSS, and vanilla JavaScript.

To preview locally:

```bash
# from the project root
# On Windows with Python installed
python -m http.server 8000
# then open http://localhost:8000
```

Publish to GitHub Pages (suggested repo name `King-Leo58148.github.io`):

1. Create a new repository named `King-Leo58148.github.io` on GitHub.
2. Initialize git, add files, commit and push to `main`:

```bash
git init
git add .
git commit -m "Initial portfolio"
git branch -M main
git remote add origin https://github.com/<your-username>/King-Leo58148.github.io.git
git push -u origin main
```

GitHub Pages will serve the site from the repository root for user/organization pages.
